# EastplayersTest
## Environment
 - Xcode 12.5.1
 - node v16.14.0
 - Android Studio 4.1.1
## Install
 - run command below
  ```
  yarn install
  cd ios && pod install
  ```
 ## Run app
  - ios: choose one of two
   ```
    yarn ios
    npx react-native run-ios
   ```
  - android: choose one of two
   ```
    yarn android 
    npx react-native run-android
   ```
